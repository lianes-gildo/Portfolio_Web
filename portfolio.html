Discente: Lianes Gildo Nhacula | Codigo de Estudante: 202301079

Introdução

Este portfólio foi desenvolvido no âmbito da disciplina de Programação Web, com o objetivo de demonstrar as competências adquiridas ao longo do semestre. O trabalho apresenta uma compilação de projetos práticos que abrangem as principais tecnologias da web moderna.

Ao longo desta disciplina, foram exploradas diversas tecnologias fundamentais para o desenvolvimento web, incluindo HTML para estruturação de conteúdo, CSS para estilização e design responsivo, JavaScript para interatividade e manipulação dinâmica de dados, e PHP para processamento do lado do servidor.

O portfólio está organizado de forma a demonstrar a evolução do aprendizado, desde conceitos básicos até implementações mais complexas, incluindo validação de formulários, manipulação de dados, criação de interfaces interativas e integração de diferentes tecnologias.

Projetos Desenvolvidos

1. Website Responsivo

Desenvolvimento de uma página web responsiva utilizando HTML5 e CSS3, com design adaptável a diferentes dispositivos e tamanhos de tela.

HTML5 CSS3

2. Formulário com Validação

Criação de formulário interativo com validação de dados em tempo real usando JavaScript, garantindo a integridade das informações inseridas.

HTML JavaScript

3. Calculadora Interativa

Implementação de uma calculadora funcional com operações matemáticas básicas, demonstrando manipulação de eventos e lógica de programação.

JavaScript CSS

4. Sistema de Navegação

Desenvolvimento de menu de navegação interativo com efeitos visuais e comportamento responsivo para melhor experiência do utilizador.

HTML CSS JavaScript

Reflexão sobre o Aprendizado:

O processo de desenvolvimento deste portfólio permitiu consolidar conhecimentos teóricos através da prática, enfrentando desafios reais de programação web. As principais dificuldades encontradas incluíram a gestão de eventos em JavaScript e a criação de layouts responsivos complexos, que foram superadas através de pesquisa, experimentação e aplicação dos conceitos aprendidos em aula.

A experiência adquirida com a validação de formulários e manipulação de dados demonstrou a importância da segurança e usabilidade em aplicações web. A integração de diferentes tecnologias (HTML, CSS, JavaScript) revelou-se essencial para criar experiências de utilizador funcionais.

Conclusão

Este portfólio representa o culminar de um semestre de aprendizado intensivo em Programação Web, demonstrando a evolução desde conceitos fundamentais até a capacidade de desenvolver aplicações web funcionais e interativas.

As tecnologias estudadas - HTML, CSS, JavaScript e PHP - formam a base essencial para o desenvolvimento web moderno. A compreensão de como estas tecnologias se complementam e interagem é fundamental para criar soluções web eficazes e eficientes.

Os projetos desenvolvidos ao longo da disciplina não apenas demonstram competência técnica, mas também revelam a capacidade de resolver problemas práticos através da programação. A validação de formulários, manipulação de dados, criação de interfaces responsivas e implementação de funcionalidades interativas são habilidades que serão aplicadas em projetos futuros.

O conhecimento adquirido nesta disciplina estabelece uma base sólida para a continuação dos estudos em desenvolvimento web, incluindo frameworks modernos, desenvolvimento back-end avançado e integração com bases de dados. A experiência prática obtida através deste portfólio será valiosa para futuros desafios profissionais e académicos na área de tecnologia da informação.
